Pokédex App

📖 Overview

This project is a simple Pokédex web application built with HTML, CSS, and JavaScript. It allows users to search for a Pokémon by name and retrieves live data from the PokéAPI.

The project was created as part of my journey into modern JavaScript development, with a focus on asynchronous programming using fetch() and async/await.

✨ Features

🔍 Search for Pokémon by name

🖼️ Display Pokémon sprite

📛 Display Pokémon name

🎨 Display Pokémon type(s)

📝 Display a Pokémon description

⚡ Fetch live data from the PokéAPI

❌ Basic error handling for invalid searches

🛠️ Built With

JavaScript (ES6+)
